Steps to install UWPTestApp.
**********************************************
Users Need to On the Developer mode from Settings Apps:

On TaskBar type Settings>>Update & security>>For Developers(under left panel)>>Enable toggle button to on the developer mode.

**********************************************
Installation Steps:
***********************************************
=>Open the setup file
=>Select Install(Right click and run as powershell option)
=>Powershell window appears=>
=>With Y/N option( press Y)
=>After creating few bundle again there is option for Y/N -Press (Y)
=> And installation process start
=> After setup is installed
=>Type UWPTestApp over StartMenu
