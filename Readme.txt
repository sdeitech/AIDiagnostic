Steps to install UWPTestApp.
**********************************************
Users Need to On the Developer mode from Settings Apps:

On TaskBar type Settings>>Update & security>>For Developers(under left panel)>>Enable toggle button to on the developer mode.

**********************************************
Installation Steps:
***********************************************
- Open the setup file.
- Right-click on install.ps.1 and choose "Run as PowerShell" to start the installation.
- Follow the prompts in the PowerShell window.
- When prompted, press "Y" to proceed.
- If given the option, press "Y" again to continue.
- The installation process will begin.
- Once the setup is installed, search for "UWPTestApp" in the Start Menu.
